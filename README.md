# Forest_Survival
___
Семестровый проект студентов 1 курса ЛФИ Губарева Егора и Кузнецовой Полины
## 1. Описание игры 
___
Проект представляет собой симулятор выживания в зимнем лесу. Главной целью игры является нахождение деревни, преодолевая такие трудности как: холод, голод, жажда и другие. Данная игра проверяет способности игрока выживания в лесу, с ограниченным количеством ресурсов. Главными достоинствами игры является то, что она завлекает и имеет неплохую графику. Возможно улучшение проекта, а именно: добавление сна в хижине, противников в виде волков и другие улучшения.
## 2. Инструкция по установке 
___
    1. Установите Python с сайта [python.org](https://www.python.org/downloads/) 
    2. Проверьте наличие библиотек pygame, random и math. В случае их отсутствия в терминале наберите:
    '''Python
    pip install pygame
    pip install random
    pip install math
    ''' 
    3. Склонируйте репозиторий к себе на компьютор. Для этого в терминале наберите:
    '''Python
    git clone https://github.com/ForestSurvival/Forest_Survival.git
    ''' 
    4. Запустите в склонированном репозитории файл main.py:
    '''Python
    cd ../Forest_Survival
    python main.py
    ''' 
    5. Запустите игру
## 3. Инструкция по использованию
___
- При запуске игры вы окажетесь в меню.
- Перед запуском игры обязательно ознакомьтесь с правилами и управлением. Там описано много нетривиальных фишек, без которых игру будет невозможно пройти.
- Для выхода в меню нажмите 'Esc'.
- Внимательно следуйте инструкциям и наслаждайтесь игрой.
